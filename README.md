# Haso Hasao 😂

A playful, mobile-first joke app with Hindi and English jokes for every age group.

## Features

- Hindi and English language selection
- Kids, teenage, adults, and old-age joke categories
- Copy, WhatsApp share, and favorite actions
- Responsive phone-style interface
- Automatic deployment to GitHub Pages

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
